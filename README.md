# CPP Assignments Repository

Welcome to the **CPP_Assignments** repository! This collection of assignments is designed to sharpen your C++ programming skills by tackling a variety of practical problems. Dive in and explore concepts ranging from basic programming constructs to object-oriented principles.

---

## 📚 Repository Contents

### **Assignment 1: Fundamental Concepts**
1. **Enums & Switch**: Display day names based on user input using an `enum` and a `switch` statement.
2. **Swapping with Reference Variables**: Swap two integers using a function with reference arguments.
3. **Dynamic Vectors**: Create a user-defined vector using the `new` operator.
4. **Matrix Operations**:
   - Dynamically allocate and read a matrix.
   - Display the matrix using functions.
   - Add default arguments for matrix dimensions.
5. **Macros & Functions**:
   - Define a macro to find the largest of three numbers.
   - Implement overloaded `power()` functions for exponentiation.
6. **Function Overloading**: Demonstrate function overloading and test with a main program.

---

### **Assignment 2: Classes and Objects**
1. **Bank Account Management**:
   - Define a class to manage a bank account (depositor’s name, account number, type, balance).
   - Add functions to deposit, withdraw, and display account details.
   - Extend to handle 10 customers.
2. **Distance Calculations**:
   - Create `DM` (meters & cm) and `DB` (feet & inches) classes.
   - Add distances using a friend function and display results in the desired format.
   - Define a `distance` class for operations like addition and display of multiple distance objects.

---

### **Assignment 3: Advanced Object-Oriented Design**
1. **String Class**:
   - Build a custom `String` class to handle uninitialized and initialized strings.
   - Add functionality to concatenate strings and copy one string to another.
2. **Bookshop Inventory**:
   - Create a `Book` class to manage inventory (author, title, price, publisher).
   - Implement search functionality for book availability.
3. **Complex Number Operations**:
   - Define a `Complex` class with real and imaginary parts.
   - Use constructors and a static member function to add complex numbers.

---

## ✨ Highlights
- **Dynamic Memory Management**: Learn to use the `new` operator efficiently.
- **Object-Oriented Programming**: Dive into classes, objects, constructors, and static functions.
- **Practical Applications**: Explore real-world scenarios like bank account management and book inventory systems.
- **Function Overloading & Macros**: Understand advanced concepts with hands-on examples.
